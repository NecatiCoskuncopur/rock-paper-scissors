# Rock-Paper-Scissors

## ➡️ [Live Demo](https://rock-paper-scissors-ecru-six.vercel.app/)

## :computer: Setup

1. Clone the repository and install its features.

```
yarn install
```

2. Use the project locally (localhost).

```
yarn start
```
